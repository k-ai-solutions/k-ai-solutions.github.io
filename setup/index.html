Codex

為 Codex CLI 新增 K-AI provider

建議新增獨立 provider，不覆蓋官方 OpenAI 預設。適合 Codex CLI、IDE extension 及 OpenAI-compatible workflow。

  1. 1安裝 Codex CLI，然後建立或打開 ~/.codex/config.toml。
  2. 2貼上 K-AI provider 設定，並把 KAI_API_KEY 設為環境變數。
  3. 3測試前重開 Terminal、PowerShell、VS Code 或你的 editor。
model = "gpt-5.4"
model_provider = "kai"

[model_providers.kai]
name = "K-AI API"
base_url = "https://api.kaihk.com/v1"
env_key = "KAI_API_KEY"
wire_api = "responses"

安全提醒

不要把 API key 發到公開群組、截圖、GitHub 或共享文件。如懷疑 key 外洩，請在 NewAPI 刪除舊 key 並建立新 key。

常見修正

  • 401：重新複製 key，確認沒有多餘空格。
  • Quota：檢查帳號是否已開通額度。
  • Model not found：選擇 NewAPI 後台可見模型。
