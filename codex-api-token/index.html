前往 NewAPI

1. 到 NewAPI 註冊

建立帳號後，到 Token / API Keys 建立自己的 API key。

2. 安裝 Codex 或 CC Switch

按 Mac / Windows 使用教學內的安裝檔或 CLI 方法。

3. 新增 K-AI API provider

Provider 名稱為 K-AI API，Base URL 是 https://api.kaihk.com/v1。

4. 重開並測試

重開 Terminal、PowerShell、VS Code 或相關工具，再做一次簡單測試。

安全提醒：不要把 API key 發到公開群組、截圖或 GitHub。如懷疑外洩，請刪除舊 key 並重新建立。

Codex CLI provider 設定

建議新增 K-AI provider，不覆蓋官方 OpenAI 預設。API key 請使用自己帳號內建立的 key。

model_provider = "kai"

[model_providers.kai]
name = "K-AI API"
base_url = "https://api.kaihk.com/v1"
env_key = "KAI_API_KEY"
wire_api = "responses"
